# Web UI Mood Map & Design System

## 1) Цель и настроение
Продукт должен ассоциироваться с женственным, мягким premium-настроением.
Референс: [stylegenie.ru](https://stylegenie.ru/).

Основные характеристики:
- **Мягкость**: скругленные углы (radius `0.75rem` - `1rem`), деликатные тени.
- **Цветовая гамма**: светлые, бледно-фиолетовые и насыщенные фиолетовые оттенки.
- **Контраст**: без агрессивных черных/белых контрастов, предпочтение отдается мягким серым и фиолетовым тонам для текста и бордеров.

## 2) Цветовые токены (Tailwind v4 / shadcn)

В проекте используется HSL-палитра для интеграции с `shadcn/ui`.

### Базовые цвета
- **Background**: `hsl(280, 40%, 98%)` — очень светлый, почти белый фиолетовый оттенок для фона страницы.
- **Foreground**: `hsl(280, 20%, 15%)` — темно-серый с легким фиолетовым отливом для основного текста.
- **Card / Popover**: `hsl(0, 0%, 100%)` — чистый белый для карточек, чтобы они выделялись на фоне.

### Акценты (Primary / Secondary)
- **Primary**: `hsl(270, 60%, 55%)` — насыщенный фиолетовый для главных кнопок (CTA).
- **Primary Foreground**: `hsl(0, 0%, 100%)` — белый текст на primary кнопках.
- **Secondary**: `hsl(280, 30%, 92%)` — бледно-фиолетовый для вторичных кнопок и плашек.
- **Secondary Foreground**: `hsl(270, 60%, 40%)` — темный фиолетовый текст на secondary элементах.

### Вспомогательные
- **Muted**: `hsl(280, 20%, 95%)` — приглушенный фон для неактивных элементов или подложек.
- **Muted Foreground**: `hsl(280, 10%, 45%)` — приглушенный текст.
- **Accent**: `hsl(280, 40%, 94%)` — акцентный фон при hover.
- **Accent Foreground**: `hsl(270, 60%, 50%)` — акцентный текст.
- **Border / Input**: `hsl(280, 20%, 88%)` — деликатные бордеры.
- **Ring**: `hsl(270, 60%, 55%)` — цвет фокуса (совпадает с primary).

## 3) Правила применения
- **Фон страницы**: использует `--background` (бледно-фиолетовый).
- **Карточки и формы**: используют `--card` (белый) с мягкой тенью (`shadow-sm` или `shadow-md`) и скруглениями (`rounded-xl` или `rounded-2xl`).
- **Кнопки CTA**: используют `--primary`, при наведении немного затемняются или получают легкую тень.
- **Состояния**:
  - *Disabled*: снижение opacity до `0.5`, без изменения цвета на серый (сохраняем оттенок).
  - *Error*: мягкий красный/розовый оттенок `hsl(340, 70%, 60%)`, не агрессивный красный.
